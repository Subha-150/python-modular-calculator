**Modular Python Calculator**
**Project Overview**__
This project is a functional command-line calculator built during my first-year internship. 
It demonstrates core Python concepts like function definition, parameter passing, and defensive programming.

Features:
Addition, Subtraction, Multiplication, and Division functions.
Default argument handling (e.g., adding zero if only one number is provided).
Error handling for division by zero to prevent crashes.
Infinite loop logic that restarts the menu even after selecting exit.
Documented functions using Python docstrings.

Internal Functions:
add(a, b=0): Returns the sum.
subtract(a, b=0): Returns the difference.
multiply(a, b=1): Returns the product.
divide(a, b=1): Returns the quotient or an error message if dividing by zero.
run_calculator(): Manages the user input and menu display.
